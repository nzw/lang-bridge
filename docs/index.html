Features

主な機能

検索・登録から学習・同期まで、語彙学習に必要なすべてを搭載。

🔍

インクリメンタル検索

曖昧・前方一致・完全一致に対応。外部辞書 API（プラグイン可能）の結果も統合表示。180 ms のデバウンスでサクサク絞り込み。

✏️

単語登録

ソース言語・ターゲット言語・メモ・カテゴリ（// 区切り）を手動入力して即登録。ホーム画面のボタンからワンタップでダイアログを開ける。

📖

マイ単語一覧

登録済み単語の絞り込み検索。一致方式・対象フィールドをアコーディオンパネルで細かく設定可能。

🃏

フラッシュカード学習

表面の言語・メモ表示面・シャッフル・お気に入りフィルタを自由に組み合わせ。学習中の戻るボタンは設定画面に戻る。

🔗

外部辞書リンク

Google・Weblio 国語・Weblio 英和/和英・ctrans.org の 4 種類を ON/OFF 設定。入力中は有効なリンクが結果先頭に表示。

📊

Google Sheets 同期

スプレッドシートからのインポート（自動/一括/シートごと）と、エクスポート（既存シート追加 / 新規作成）をサポート。

🗂️

データ管理

取込セッション単位・手動登録・全データの削除が可能。設定画面でソース別の件数と最終更新日時を確認できる。

☁️

Google アカウント連携

OAuth 2.0 で Google ログイン。ログイン済みアカウントは全画面で共有され、同期時に自動で認証が使われる。

Dictionary Links

外部辞書リンク設定

検索ボックスに文字を入力すると、有効なリンクが結果リストの先頭に表示されます。 設定 → 辞書設定 で ON/OFF を切り替え、SharedPreferences に保存されます。

Import / Export

Google Sheets 同期

スプレッドシートのインポートとエクスポートを、ログイン済みアカウントで安全に処理。

📥 インポート

1
モード選択

自動 / 一括 / シートごと の 3 モード

2
列マッピング

標準ヘッダー（ソース言語/日本語/ターゲット言語/中国語）は自動判定。未知のヘッダーのみダイアログ表示。

3
重複処理

上書き または 新規追加のみ を選択

4
カテゴリ自動付与

シート名が先頭カテゴリとして自動追加される

📤 エクスポート

📄 既存スプレッドシートに新規シート追加
URL を貼り付けて既存シートに追記。新規シート名デフォルト: yy/mm/dd（実行日）
✨ 新規スプレッドシート作成
新規作成。タイトルデフォルト: LangBridge Export yyyymmdd
✅ 完了後
生成された URL をタップしてブラウザで開くか、コピーボタンでクリップボードに保存
